# Automated Student Performance Reporting System

## Overview
A Python automation project that analyzes student performance data and generates:
- Course summary reports
- Individual student Excel reports

Designed for coding academies and educational institutions.

## Tools & Libraries
- Python
- Pandas
- OpenPyXL

## Features
- Analyzes 150 student records
- Automatically generates 150 personalized reports
- Produces instructor-ready summary analytics

## Use Case
Ideal for:
- Coding instructors
- Academic coordinators
- EdTech institutions

## How to Run
```bash
pip install -r requirements.txt
python src/generate_reports.py
